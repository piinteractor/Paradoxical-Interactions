Science & Académie

Quand le contrôle qualité empêche la qualité. Quand le scepticisme devient dogme. Quand le système de production de connaissances bloque la connaissance.

Le Paradoxe de Galilée

La structure :

L'Église a développé des méthodes rigoureuses pour évaluer les prétentions à la vérité – Écriture, tradition, consensus théologique. Ces méthodes ont fonctionné pendant des siècles. Galilée présente des preuves contredisant la cosmologie établie. Les preuves sont empiriques. La méthode est théologique. Le système ne peut accepter les preuves sans invalider la méthode qui définit le système.

Pourquoi chaque acteur est rationnel :

Autorités ecclésiastiques : - Défendent l'épistémologie établie (leur mandat)

Théologiens : - Appliquent des méthodes éprouvées (siècles de tradition)

Galilée : - Présente des preuves empiriques (méthode scientifique)

Aristotéliciens : - Défendent un cadre philosophique cohérent (pas stupides, juste paradigme différent)

Pourquoi ça échoue collectivement :

La rationalité de l'Église (défendre l'autorité doctrinale) et la rationalité de Galilée (suivre les preuves empiriques) sont incompatibles dans le même système. Accepter Galilée signifie invalider la prétention de l'Église à arbitrer la vérité. Rejeter Galilée signifie ignorer les preuves. Les deux choix détruisent quelque chose d'essentiel.

Le piège : Le système conçu pour préserver la vérité ne peut reconnaître une nouvelle vérité sans se détruire lui-même.

Gardiennage par Évaluation par les Pairs

La structure :

Les revues utilisent l'évaluation par les pairs pour assurer la qualité. Les évaluateurs sont des experts du domaine. Les experts ont des positions établies, des cadres, des carrières construites sur les paradigmes actuels. Les soumissions remettant en question ces paradigmes sont évaluées par des personnes investies dans leur validité. Les idées radicales échouent à l'évaluation non pas parce qu'elles sont fausses, mais parce qu'elles sont trop différentes.

Pourquoi chaque acteur est rationnel :

Revues : - Veulent un contenu crédible (la réputation compte)

Évaluateurs : - Appliquent les normes du domaine (c'est l'expertise)

Éditeurs : - Font confiance au jugement d'expert (qui d'autre peut évaluer ?)

Chercheurs établis : - Défendent des cadres qui fonctionnent (pas malveillants, protecteurs)

Organismes de financement : - Soutiennent la recherche évaluée par les pairs (où allouer autrement ?)

Pourquoi ça échoue collectivement :

Le système optimise pour des avancées incrémentales dans les paradigmes acceptés. Les changements de paradigme – par définition – violent les normes actuelles. Les gardiens ne sont pas biaisés, ils sont compétents. Leur compétence est le problème.

Le piège : Le contrôle qualité devient contrôle de l'innovation.

Publier ou Périr

La structure :

L'académie mesure la productivité par le nombre de publications. Les chercheurs optimisent pour les publications. Plus d'articles, découvertes plus petites. Les revues prolifèrent. La qualité se disperse. Les évaluateurs sont submergés. La qualité de l'évaluation baisse. Les mauvais articles sont publiés. La confiance dans les publications s'érode. La pression pour publier augmente (pour se démarquer du bruit). Le cycle s'accélère.

Pourquoi chaque acteur est rationnel :

Chercheurs : - Publient pour survivre (exigence de carrière)

Universités : - Mesurent la production (comment évaluer autrement ?)

Revues : - Publient plus (modèle économique)

Évaluateurs : - Survolent les articles (trop nombreux pour une lecture approfondie)

Administrateurs : - Comptent les publications (métrique objective)

Pourquoi ça échoue collectivement :

Inciter la quantité détruit la qualité. Tout le monde le sait. Personne ne peut arrêter. Le chercheur qui publie moins se voit refuser la titularisation. La revue qui rejette plus reçoit moins de soumissions. L'université qui change les métriques perd dans les classements.

Le piège : La mesure du succès empêche le succès.

Crise de Réplication

La structure :

La science récompense les découvertes nouvelles. Les revues publient les résultats positifs. Les résultats nuls restent non publiés (pas intéressants). Les chercheurs mènent des études, ne trouvent rien, classent. Quelqu'un d'autre mène une étude similaire, ne trouve rien, classe. Finalement quelqu'un obtient un faux positif (les statistiques le garantissent). Cela est publié. Personne ne publie les vingt résultats nuls. La littérature montre « l'effet existe ». Il n'existe pas.

Pourquoi chaque acteur est rationnel :

Chercheurs : - Publient des découvertes nouvelles (c'est ainsi qu'on obtient du financement)

Revues : - Publient des résultats intéressants (c'est ce que veulent les lecteurs)

Organismes de financement : - Soutiennent la recherche prometteuse (pas « nous n'avons rien trouvé »)

Médias : - Rapportent les percées (les résultats nuls ne sont pas des nouvelles)

Universités : - Promeuvent les découvertes réussies (valeur PR)

Pourquoi ça échoue collectivement :

Le biais de publication crée un faux consensus. Le système sélectionne les faux positifs. La réplication échoue, mais les échecs ne sont pas publiés (toujours pas intéressants). La structure produit des connaissances peu fiables pendant que tout le monde suit les règles.

Le piège : Le système pour découvrir la vérité produit la fausseté.

Spécialisation Académique

La structure :

Le savoir se développe. Les chercheurs se spécialisent pour contribuer de manière significative. La spécialisation s'approfondit. Les sous-domaines se fragmentent. Les spécialistes ne peuvent pas communiquer à travers les frontières des sous-domaines. Les problèmes interdisciplinaires (climat, IA, santé publique) nécessitent l'intégration. Personne n'a l'ampleur. Les généralistes manquent de profondeur. Les spécialistes manquent de contexte.

Pourquoi chaque acteur est rationnel :

Chercheurs : - Se spécialisent pour être compétitifs (la profondeur gagne les subventions)

Départements : - Embauchent des spécialistes (expertise mesurable)

Conférences : - S'organisent par sous-domaine (discussions cohérentes)

Revues : - Se concentrent sur des niches (portée plus claire)

Étudiants : - Se forment étroitement (employabilité)

Pourquoi ça échoue collectivement :

Les vrais problèmes ne respectent pas les frontières disciplinaires. La structure qui produit l'expertise empêche la synthèse. Les spécialistes résolvent brillamment des problèmes étroits. Les grands problèmes restent non résolus – non pas parce que les solutions sont inconnues, mais parce que personne ne peut intégrer les pièces.

Le piège : La profondeur des connaissances crée l'aveuglement.
