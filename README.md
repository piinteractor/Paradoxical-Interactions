# Paradoxical Interactions (PI)

**A framework for understanding why intelligent actors consistently create the disasters they're trying to prevent.**

## Overview

Paradoxical Interactions (PI) emerge when every participant acts rationally from their own perspective, yet collectively they generate catastrophe. Not through stupidity or malice, but structurally.

**All are guilty. None are at fault.**
