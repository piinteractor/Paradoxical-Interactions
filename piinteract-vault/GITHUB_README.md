## What is Paradoxical Interactions?

**PI is not a theory. Just a little thinking model. But it works. Devilishly good.**

Paradoxical Interactions (PI) is a framework for understanding structural situations where rational individual actors consistently produce collectively irrational outcomes.

**Core Principle:** *"All are guilty, none are at fault"*
